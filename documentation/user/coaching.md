# Coaching & the dashboard

Coaches maintain the strat library. Your team shares a **coach password** (set by whoever runs the site). That unlocks the coach dashboard — there is no separate email login in the app.

## Open the coach area

1. Click **Coach** in the header.  
2. Enter the **coach password** on the login screen.  
3. Use **Lock coach** when you’re done on a shared computer.

## Create a strat

Fill in the form:

| Field | What to put |
|-------|-------------|
| **Title** | Clear round name (e.g. “A default — fast plant”) |
| **Map** | Map name as you want it to appear in filters |
| **Side** | Attack or Defense |
| **Difficulty** | 1–3 |
| **Agents** | Comma-separated (e.g. `Jett, Sova, Omen`) |
| **Summary** | What players see first on the card |
| **Round plan** | One step per line; you can use `<strong>Agent</strong>` for emphasis |
| **Roles** | One line per role, format: agent name, then a pipe, then the job (same as the form hint) |
| **Coach notes** | Timings, contingencies, comms — shown in the detail view |
| **Tags** | Comma-separated; helps search |
| **Images** | Paste image URLs, or **Upload** while signed in (stored for the team) |

Submit **Create strat**. It appears on **Browse** for everyone immediately.

## Edit or delete

Under **Your strats**, use **Edit** to load a strat into the form, change fields, then **Save changes**. **Delete** removes it permanently — there is no undo in the app.

## Sign out

Use **Sign out** on the coach page when you’re done on a shared computer.
