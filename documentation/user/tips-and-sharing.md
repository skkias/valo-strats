# Tips & sharing

## ValoPlant and other images

Many teams draw executes in **ValoPlant** (or similar tools), take a **screenshot**, then either:

- Upload through the coach form (while signed in), or  
- Host the image somewhere your team trusts and paste the **URL** into an image row.

Players see those images in the strat detail view.

## Discord and links

The Browse page has a normal web address (for example your team’s Vercel URL). You can:

- Paste that link in Discord so people open the **full app**  
- Link to a specific strat by sharing what you see in the browser after you open a card (if your team bookmarks deep links)

**Note:** Discord’s own “embed” cards are not the same as this app; for rich layouts, Discord uses its own format. For your drawn plans, linking to this site or posting images in Discord both work well.

## Phones and tablets

The layout works on small screens: filters stack, the file tree for **Documentation** moves below the article, and strat cards stay easy to tap.

## Documentation lock

If **Documentation** asks for a password, that’s set by whoever runs the site — it’s only for these help pages, not for viewing strats on **Browse**.
