# Documentation layout

This folder splits **who the docs are for**:

| Path | Audience | Served in the app? |
|------|----------|--------------------|
| **`user/`** | Players and coaches using Valo Strats | Yes — password-protected **Documentation** tab (`/docs`) |
| **`developer/`** | Engineers maintaining the repo | No — read/edit in the editor or on GitHub |

- User guides use plain language and only describe in-app behavior.
- Developer notes cover stack, env vars, architecture, and how the docs viewer works.
